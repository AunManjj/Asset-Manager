AUTOMATION
===========================================================

Use n8n Community Edition (Free).

Do NOT build a custom scheduler.

Instead, prepare the backend and provide n8n workflow definitions.

Inside backend/workflows/

Create JSON workflow exports for:

1. Hourly Meta Ads Sync
   - Cron trigger (every hour)
   - Fetch campaigns, ad sets, ads, insights
   - Save to PostgreSQL

2. Daily KPI Check
   - Every morning
   - Compare KPIs
   - Generate OpenAI summary
   - Send Slack notification

3. Lost Call Analysis
   - Webhook trigger
   - Receive call notes
   - OpenAI analyzes objections
   - Save AI insight
   - Send Slack message

4. Weekly PDF Report
   - Weekly Cron
   - Generate PDF
   - Email client
   - Save report
   - Notify Slack

5. Campaign Performance Monitor
   - Every 4 hours
   - Compare last 28 days
   - Detect anomalies
   - AI explanation
   - Slack alert

Document how to import these workflows into n8n Community Edition.
